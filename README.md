# Binge — Waitlist Landing Page

Static landing page for Binge (bingemicrodrama.com). Deployed via GitHub Pages.

## Files
- `index.html` — main page
- `styles.css` — all styles
- `binge_icon.png` (and resized variants) — Binge logo
- `CNAME` — custom domain mapping

## Email capture
Form uses Formspree. Update the `FORMSPREE_ENDPOINT` constant at the bottom of `index.html` with your live form ID. Submissions go to joinbinge@gmail.com.
